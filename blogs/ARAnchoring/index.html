Anchoring Virtual Objects to Specific Cards

By Setareh Soltanieh

Visualizing Different AR Assets on Playing Cards

This project demonstrates how to anchor virtual objects to specific playing cards using Unity's ARTrackedImageManager. The application, developed for Android, enables users to see a virtual robot appear on the Queen of Hearts and a virtual house on the Jack of Hearts when viewed through the deviceâ€™s camera.

How It Works

  1. Download and install the application (.apk file linked below).
  2. Grant camera permissions when prompted.
  3. Point your device at the provided playing card images.
  4. Watch as the corresponding AR object appears on the screen.
