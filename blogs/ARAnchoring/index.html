Anchoring Virtual Objects to specific Cards

Setareh Soltanieh

Visualizing Different AR Assets on Different Cards

In this project, I anchored different AR assets to different cards using the ARTrackedImageManager in Unity. This application is developed for Android users and its.apk file is provided. In this application, a virtual robot is anchored to the Queen of hearts card provided here and a virtual house is anchored to the Jack of hearts card provided here. Whenever these images are available in the camera frame, the user can see these objects. Here is a demonstration of this application.

If you want to try this application yourself, you can download this application using the following .apk file.

Download APK
