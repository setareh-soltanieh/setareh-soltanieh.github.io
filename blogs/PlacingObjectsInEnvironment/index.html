Setareh Soltanieh

Placing a vase in the environment

In this project I developed an AR application that is able to detect flat surfaces and feature points in the environment. Feature points are typically notable features in the environment that the device can track between frames, such as a knot in a wooden table. Using these detected feature points and flat surfaces, the user is able to place a virtual vase in their environment. This application uses the ARPointCloudManager and PlaneDetection packages of Unity.
