Flarpy Blorb Game

By Setareh Soltanieh

I have developed a 2D game in Unity inspired by the classic Flarpy Blorb gameplay. Built using the Unity Engine, this game challenges players to navigate a series of obstacles by timing their jumps perfectly. With smooth controls, engaging mechanics, and increasing difficulty over time, it offers a fun and addictive experience for players of all ages. The game features dynamic obstacle generation, responsive physics, and a progressive speed increase to keep the gameplay exciting.
